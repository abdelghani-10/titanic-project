# Titanic Data Preprocessing & Modeling

This project implements a complete data preprocessing and modeling pipeline using the Titanic dataset from Seaborn.  
The goal is to clean the data, engineer new features, encode categorical variables, scale numerical features, train machine learning models, and evaluate their performance.
